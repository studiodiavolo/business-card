# Studio Diavolo – Web Business Card

This project is a web-based business card created using HTML and CSS. The goal was to translate my personal branding into a digital format while keeping the design consistent with my visual identity.

The card is inspired by tarot cards. The front features an illustration, and the back includes my name, role, contact information, and social links. I used a flip animation to make the interaction more engaging.

## Purpose

The project delivers a digital version of a business card that feels more interactive than a traditional printed one. It solves the problem of presenting personal branding in a way that is both visual and functional on the web.

## Features

- Flip card animation using CSS  
- Custom layout based on a Figma mockup  
- SVG icons for social media links  
- Star background created with CSS  
- Responsive proportions using aspect-ratio  

## Process

I started by creating a high-fidelity mockup in Figma and organizing all my assets. Then I built the HTML structure and used CSS to position each element and match the design.

One of the main challenges was aligning everything using absolute positioning. I also had to work with SVG icons and learn how to style them using CSS. Another challenge was making the flip animation smooth and making sure both sides of the card aligned correctly.

## Branding

This project is based on my personal brand, Studio Diavolo, which I have been developing over time. My style focuses on bold, expressive visuals with inspiration from tarot cards and illustration. This project helped me translate that identity into a web format.

## Typography

I used my selected brand fonts to keep consistency with my identity. The typography was chosen to match the tarot-inspired aesthetic while still being readable on screen.

## Accessibility

I added alt text to images to improve accessibility and make the content more understandable. I also paid attention to color contrast to ensure the text is readable against the background.

## What I Learned

This was one of my first complete web projects. I learned how to translate a visual design into HTML and CSS, especially how positioning, typography, and interaction work together.

I also learned how to use CSS for animations and how to work with SVG elements.

## Notes

What makes this project stand out is the combination of illustration, personal branding, and interaction. Instead of a static layout, the flip animation creates a more engaging experience while still keeping the design clean and focused.
